# ☕ Caffeine Cove

Caffeine Cove is a modern and cozy coffee shop website built with HTML and CSS.  
The project focuses on a clean design, smooth layout, and a warm coffee-shop atmosphere.

---

## 🌟 Features
- Responsive design (desktop & mobile)
- Modern coffee shop UI
- Clean HTML & CSS structure
- Smooth navigation sections
- Image-based layout

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Font Awesome Icons
